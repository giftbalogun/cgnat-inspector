# Screenshots

This directory holds screenshots referenced by the main `README.md`
(e.g. `demo.png`, `json-output.png`). Add PNG/GIF captures of:

- `cgnat-inspector` default (human-readable) output showing a CGNAT
  detection result
- `cgnat-inspector` output showing a `PUBLIC` result, for contrast
- `cgnat-inspector --json | jq .` output
- `cgnat-inspector --verbose` showing intermediate test detail

Keep images reasonably sized (compress PNGs, prefer < 500KB each) since
they're committed directly to the repository.
